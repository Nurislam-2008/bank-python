# bank-python 
Создал шаблон (Класс BankAccount) Описал, как выглядит счет имя владельца и баланс и что он умеет делать (пополняться, снимать деньги и показывать данные).
Добавил проверки (например, нельзя снять больше, чем есть на счету, или положить отрицательную сумму).
Организовал хранение данных Используется словарь accounts, чтобы программа помнила все созданные счета по именам.
Сделал интерфейс: Написал бесконечный цикл while с меню, который позволяет пользователю взаимодействовать с программой через консоль.
Создавать новые счета можно указать имя владельца и начальную сумму.
Переключаться между аккаунтами можно выбрать конкретный счет для работы.
Пополнять баланс и снимать наличные: внутри стоят проверки, чтобы нельзя было уйти в минус.
Просматривать статистику: можно увидеть баланс одного счета или список всех зарегистрированных пользователей.
